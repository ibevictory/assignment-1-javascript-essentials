There are four ways we can access elements in the DOM (Data Object Model) which are;

  1. getElementById - It uniquely identifies the element with the id name specified unlike tag, class and name id property is unique cause an id could only be used once
  2. getElementsByClassName - It returns an array of elements. and more than one element can have the same class name
  3. getElementsByTagNames - It returns an array of elements based on the tag name
  4. getElementsByName - This function selects an array of elements based on the name
